tutorizeR

tutorizeR est un package R pour convertir du matériel pédagogique existant en R Markdown (.Rmd) ou Quarto (.qmd) en tutoriels interactifs pour learnr ou quarto-live. Il permet de conserver une source de cours principale tout en générant des exercices, des solutions, des questions à choix multiples et des rapports de conversion.

Version CRAN : 0.4.5, publiée le 11 juin 2026.

Exemple d’utilisation

Le code suivant reprend le démarrage rapide du package. Il copie le module d’exemple installé, charge une banque de questions et convertit une leçon Quarto en tutoriel learnr.

# Load library
library(tutorizeR)

# Locate the installed example module
example_dir <- system.file(
  "examples",
  "example_course_module",
  package = "tutorizeR"
)

# Copy the lesson and data to a temporary working directory
work_dir <- file.path(tempdir(), "tutorizeR-example")
dir.create(work_dir, recursive = TRUE, showWarnings = FALSE)

file.copy(
  from = file.path(example_dir, "lesson-source.qmd"),
  to = work_dir,
  overwrite = TRUE
)

file.copy(
  from = file.path(example_dir, "student_activity.csv"),
  to = work_dir,
  overwrite = TRUE
)

# Load the reusable question bank
question_bank <- load_question_bank(
  file.path(example_dir, "question-bank")
)

# Convert the lesson to an interactive tutorial
report <- tutorize(
  input = file.path(work_dir, "lesson-source.qmd"),
  output_dir = work_dir,
  format = "learnr",
  assessment = "both",
  question_bank = question_bank,
  mcq_source = "mixed",
  overwrite = TRUE,
  verbose = FALSE
)

# Inspect the conversion report
print(report)

À quoi sert tutorizeR ?

  • convertir des leçons .Rmd ou .qmd en tutoriels interactifs ;
  • transformer des blocs de code existants en exercices, solutions et rétroactions ;
  • intégrer des questions à choix multiples inline ou à partir de banques YAML/JSON ;
  • produire des rapports de conversion JSON ou YAML pour révision et reproductibilité ;
  • traiter un dossier complet de leçons avec une politique de conversion cohérente.
